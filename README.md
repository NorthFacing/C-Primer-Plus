怎么学习C语言指针：
1:指针数组是一个数组，它的元素是一个指针
2:数组指针是一个指针，它指向数组的首地址
3:指针函数是一个函数，它的返回值是一个指针
4:函数指针是一个指针，它的指向函数的入口地址。
